## 🚀 Quick Start

### Installation

```bash
pip install marketing_spec_kit
```

### Basic Usage

```bash
# Initialize a new specification
marketing_spec_kit init my-spec.md

# Validate specification
marketing_spec_kit validate my-spec.md

# Generate implementation
marketing_spec_kit generate my-spec.md --output ./output
```

## 📋 Core Entity: Spec

This toolkit revolves around the `Spec` entity:

- **name** (`string`): Spec name
- **version** (`string`): Spec version
- **description** (`string`): Spec description
- **status** (`string`): Spec status (draft, review, approved)
