## ğŸ†˜ Troubleshooting

### "AI doesn't follow the two-layer architecture"
- Remind AI to read `AGENTS.md` for complete guidance
- Check that command files are present in `.marketingspeckit/commands/`

### "Generated code is incomplete"
- Ensure `/marketspec.tasks` was run before `/marketspec.implement`
- Check that `tasks.md` has sufficient detail
- Verify `src/campaigns/*.ts` was generated

### "Review command has no data"
- Verify campaign code has been executed
- Check `data/` for collected data files
- If no data, AI should prompt user for manual input
